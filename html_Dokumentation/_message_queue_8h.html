uint8_t OS_MsgQ_ReadTimeout ( OS_MessageQueue_t * pQ,
uint32_t * pDest,
uint32_t u32TimeoutTicks )

Liest eine Nachricht aus der Queue mit Timeout.

Versucht zu lesen. Ist die Queue leer, blockiert der Task maximal für die angegebene Zeit. Erfolgt in dieser Zeit kein Schreibvorgang, kehrt die Funktion mit einem Timeout-Fehler zurück.

Parameters
pQZeiger auf die Message Queue.
pDestZeiger auf die Zielvariable für die gelesene Nachricht.
u32TimeoutTicksMaximale Wartezeit in OS-Ticks.
Returns
1 bei erfolgreichem Lesen, 0 bei einem Timeout.

void OS_MsgQ_SendBlocking ( OS_MessageQueue_t * pQ,

Sendet eine Nachricht in die Queue (Blocking).

Wenn die Queue voll ist, wird der aufrufende Task blockiert und in der Bitmaske u32WaitingWriters registriert. Er wird erst wieder fortgesetzt, wenn ein anderer Task eine Nachricht ausliest und Platz schafft.
