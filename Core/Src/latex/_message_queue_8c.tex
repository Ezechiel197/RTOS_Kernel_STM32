\doxysubsection{Detailed Description}
Implementierung einer Message Queue für die Inter-\/\+Task-\/\+Kommunikation. 

Diese Datei enthält die Implementierung einer threadsicheren Message Queue basierend auf einem Ringpuffer (Circular Buffer). Sie bietet Mechanismen für Non-\/\+Blocking, Blocking und Timeout-\/basiertes Senden und Empfangen von Nachrichten. Die Synchronisation erfolgt über globale Interrupt-\/\+Sperren und das Auslösen eines Pend\+SV-\/\+Interrupts für den Context Switch.
